Operations

The Met.Office's day-to-day forecasting is under the auspices of the World Meteorological Organisation's World Weather Watch programme. Observations are made at thousands of sites world wide on an hourly basis, 24 hours per day, and the data exchanged globally. Data is also collected from ships, aircraft, radars and satellites. The data is exchanged in agreed, language independent formats. The following links give information about these formats.

BUFR Table B

Classification table of elements. Table B is referenced in section 3 of a BUFR message and contains descriptors of parameters encoded in section 4.

BUFR Table D

Lists of common sequences. Table D consists of sequences of descriptors which are expected to be in common use.

BUFR Code and Flag Table

Some meteorological parameters are best represented with reference to a code table. The BUFR Code and Flag Table refers to elements defined within BUFR Table B.

Global Master Station List (8.2 MBytes)

Details of global observing stations, both surface and upper air.
